# 🌐 Mi portafolio personal

## 👩‍💻 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un portafolio web utilizando HTML y CSS. El objetivo es presentar información sobre mí, mis habilidades, proyectos y gustos personales, aplicando buenas prácticas de maquetación semántica, diseño responsivo y estilización moderna.

El sitio incluye secciones como:
- 👤 Presentación personal  
- 📁 Tarjetas de Proyectos  
- 🧠 Habilidades  
- 🎬 Películas favoritas  
- 📬 Formulario de contacto 

Logrando una experiencia visual atractiva e intuitiva.
